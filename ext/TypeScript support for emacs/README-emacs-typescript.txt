emacs TypeScript Readme
----------------------------------------------------------------------------

TypeScript bundle for emacs, this bundle provides syntax highlighting and indenting.

This is provided as a sample to help you get started coding with TypeScript in emacs

## Feature
1. TypeScript language syntax highlighting
2. TypeScript indenting
